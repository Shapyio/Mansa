## ⚙️ GitHub Actions (CI)

`.github/workflows/build.yml`:

```yaml
name: C++ CI Build

on: [push, pull_request]

jobs:
  build:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v3
      - name: Install deps
        run: sudo apt install -y g++ cmake make
      - name: Configure
        run: cmake -S . -B build
      - name: Build
        run: cmake --build build
      - name: Run tests
        run: ./build/tests/unit_tests
```
